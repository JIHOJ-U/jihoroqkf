# JOE.DEV - 개발 외주 포트폴리오 사이트

풀스택 개발자 포트폴리오 + 외주 문의 관리 플랫폼

## 기술 스택

**Frontend**
- React 19
- React Router DOM
- Framer Motion
- Axios
- React Icons

**Backend**
- Node.js + Express
- Multer (파일 업로드)
- JSON 파일 기반 데이터 저장

## 주요 기능

- 인터랙티브 Hero 섹션 (물리 시뮬레이션 공 놀이)
- 포트폴리오 CRUD (이미지 업로드, 카테고리 필터)
- 외주 프로젝트 문의 폼
- About 페이지 (Ball Pit 미니게임, 27가지 기술 스택)
- 커서 팔로워 + 플로팅 파티클 애니메이션
- 반응형 디자인 (모바일/태블릿/데스크탑)

## 실행 방법

### Backend (port 3001)
```bash
cd backend
npm install
npm start
```

### Frontend (port 3000)
```bash
cd frontend
npm install
npm run dev
```
